# Selenium and API Testing – Practice Repository

This repository contains **practice exercises** for **Selenium UI automation, API testing, and pytest** using Python.

It is maintained **purely for learning and experimentation**.

---

## Scope

- Selenium WebDriver (Python)
- API testing with `requests`
- pytest basics (fixtures, parametrization, assertions)
- Exploratory automation exercises

> This is **not** a production-ready framework.  
> Code structure and patterns evolve as part of the learning process.

## Tech Stack

- Python
- Selenium
- pytest
- requests
